# FFmpegGuiTool.exe
使用python开发的，基于FFmpeg.exe开发的可视化视频分片处理工具，目前支持四种转码格式
- 720P-2M
- 720P-4M
- 1080P-2M
- 1080P-4M

- 2、填写输出路径
- 3、选择编码格式

- 6、等待任务处理完成即可
