# FFmpegGuiTool.exe
使用python开发的，基于FFmpeg.exe开发的可视化视频分片处理工具，目前支持四种转码格式
- 720P-2M
- 720P-4M
- 1080P-2M
- 1080P-4M
## <食用>方法
![image](https://github.com/univspeed/FFmpegGuiTool/assets/23521638/6da2e6cd-0993-4df4-be7b-799ffeadd654)
- 1、填写输入文件
- 2、填写输出路径
- 3、选择编码格式
- 4、点击开始转换
- 5、如需要查看滚动日志，勾选自动滚动日志
- 6、等待任务处理完成即可
